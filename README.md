# Dino Calm

Dino Calm is an Expo-based mobile app MVP designed to help users release stress through a cute dinosaur companion.

The project combines emotional interaction, calming guidance, and a simple growth reward system. Users can complete small calming actions, receive XP, and gradually build a positive emotional routine with their dinosaur companion.

## Core Idea

Many students and young people experience stress, anxiety, and emotional fatigue in daily life.  
Dino Calm aims to provide a lightweight and friendly digital companion that helps users calm down through small, repeatable actions.

Instead of making stress management feel serious or difficult, the app uses a warm dinosaur character, simple interaction, and positive rewards to make emotional recovery easier and more approachable.

## Main Features

- Dinosaur companion interface
- Mood check-in design
- Calming interaction flow
- XP and reward mechanism
- Simple mobile-first UI
- Expo-based rapid prototype

## Tech Stack

- Expo
- React Native
- TypeScript

## MVP Progress

The current MVP has completed the basic app structure, main interface, and initial interaction logic.  
The project is still under active development and will continue to improve in design, function, and user experience.

## Future Development

- More dinosaur states and animations
- Daily emotional check-in records
- Breathing guidance module

- Better visual design
- Possible App Store / Android release in the future

## Project Vision

Dino Calm hopes to become a small but meaningful emotional support tool.  
It is not only an app, but also a warm digital companion that encourages users to take care of their mental state step by step.
